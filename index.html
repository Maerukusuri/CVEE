Enesetuvustus

Praktilise kogemusega QA insener, kes on tegelenud reaalsete kliendiprojektide testimise ja ärikinnisvara platvormi toetamisega. Mul on kogemus funktsionaalse, regressioon-, suitsu-, kasutusmugavuse, API ja andmebaaside testimisega, samuti testdokumentatsiooni loomise, vigade raporteerimise ning nõuete analüüsiga. Olen vilunud veebirakenduste, staging-keskkondade ja sisuhaldussüsteemide (CMS) testimises. Mind iseloomustab suur tähelepanu detailidele, kasutajakeskne mõtlemine ja meeskonna toetamise kogemus, mille sain QA praktikantide ja alustavate tiimide juhendamisel. Hetkel arendan aktiivselt oma teadmisi testiautomaatika valdkonnas (Playwright, TypeScript ja Python) ning otsin uusi väljakutseid QA insenerina ühtses ja toote arendamisele suunatud tiimis.

Tehnilised oskused

Testimise põhioskused: Manuaaltestimine, funktsionaalne testimine, regressioontestimine, suitsutestimine, kasutusmugavuse testimine, UI testimine, API ja DB testimine, DevTools, testlood, kontrollnimekirjad, vigade raporteerimine.
Tööriistad ja automaatika: Playwright, TypeScript, Python, POM (Page Object Model), Git, Postman, Swagger, DBeaver, SQL, WordPress, GeoJSON, YouTrack, Agile.
Pehmed oskused: Tiimidevaheline koostöö, mentorlus ja juhendamine, nõuete analüüs, probleemide lahendamine, selge kommunikatsioon.
